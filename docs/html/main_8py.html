Detailed Description

Programa interativo para operações sobre um vetor de números inteiros.

Function Documentation

◆ ler_vetor()

list[int] main.ler_vetor ( )

Pede ao utilizador para inserir números inteiros válidos e devolve o vetor.

Esta função pede ao utilizador para inserir números inteiros entre NUMERO_MIN e NUMERO_MAX até que o vetor atinja o tamanho TAMANHO_VETOR. Números inválidos são rejeitados com uma mensagem de erro.

Returns
Lista de números inteiros válidos.

◆ main()

main.main ( )

Função principal do programa.

Esta função pede ao utilizador números inteiros e guarda-os num vetor, para posteriormente providenciar forma de fazer operações sobre esses valores.
